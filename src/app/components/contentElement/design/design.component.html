Elección de colores

La aplicación cuenta con una amplia selección de colores los cuales pueden ser fácilmente cambiados por el usuario. Gracias a la facilidad que ofrece Flutter para cambiar el theme de la aplicación, Natrium se encargó de ofrecer varias opciones, las cuales son las siguientes: 

static const brightBlue = Color(0xFFA3CDFF); // Natrium
static const blueishGreen = Color(0xFF61C6AD); // Titanium 
static const deepBlue = Color(0xFF0050BB); // Indium 
static const blue = Color(0xFF4A90E2); // Neptium 
static const teal = Color(0xFF75F3FF); // Thorium

Elección de estilos de texto

Respecto al estilo del texto, ellos tomaron la decisión de utilizar tres familias de texto: Nunito Sans, Comfortaa y OverpassMono los cuales varían su tamaño de acuerdo al widget que lo requiera. El color cambia de acuerdo al tema que el usuario tenga escogido.
